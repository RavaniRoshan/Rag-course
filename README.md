# Academic RAG Journal

[![Website](https://img.shields.io/badge/Website-rag--course--two.vercel.app-blue)](https://rag-course-two.vercel.app/)

A high-fidelity, distraction-free reading environment for mastering Retrieval Augmented Generation. This digital textbook treats technical documentation with the typographic reverence of a printed academic journal.

**View the course online at [rag-course-two.vercel.app](https://rag-course-two.vercel.app/)**

---

## Course Overview

This course provides an in-depth exploration of RAG systems with practical, testable code examples. Designed for AI Engineers and Data Scientists who value deep work and need a calm, authoritative space to internalize complex architectural patterns.

**Inspired by:** *The New Yorker's* long-form layout, *Stripe Press* books, *Distill.pub*.

---

## Course Modules

1. **Module 1 - Reranking**: Two-Step Retrieval with Cross-Encoders
2. **Module 2 - Chunking Strategies**: Document Segmentation techniques
3. **Module 3 - Embedding Models**: Selection and evaluation for specific domains
4. **Module 4 - Vector Databases**: Storage and optimization strategies
5. **Module 5 - Hybrid Search**: Combining semantic and keyword search
6. **Module 6 - Query Understanding**: Transformation and improvement techniques
7. **Module 7 - Multi-Query Expansion**: Generating better search queries
8. **Module 8 - Context Compression**: Reducing context length while preserving relevance
9. **Module 9 - Evaluation Metrics**: Precision, recall, MRR, and RAG performance
10. **Module 10 - Metadata-Enhanced Retrieval**: Using metadata to improve search
11. **Module 11 - Agentic RAG Workflows**: Advanced autonomous systems
12. **Module 12 - Optimization Techniques**: Latency reduction and cost optimization
13. **Module 13 - Domain Adaptation**: Customizing RAG for specific industries
14. **Module 14 - Advanced Architectures**: Self-RAG, CRAG, and sophisticated approaches
15. **Module 15 - Production Deployment**: Scaling and maintaining RAG systems

## Running Examples

Each module contains testable code examples in the `modules/` subdirectory:

```bash
cd modules/module_01_reranking
python example.py
```

## Design Philosophy

**"The Digital Monograph"** - Warm, sepia-toned backgrounds, classical serif typography, and generous whitespace. It feels like reading a well-typeset paper on an iPad Pro, but on the web.

- **Primary**: Deep Ink (`#2C2420`) - Body text, primary headers
- **Background**: Old Paper (`#F9F7F1`) - Main page background
- **Surface**: Parchment (`#EBE6DA`) - Card backgrounds, code blocks
- **Accent**: Editorial Red (`#9E2A2B`) - Active links, highlights
